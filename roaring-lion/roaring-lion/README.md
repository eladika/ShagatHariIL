# 🛡️ מדריך שאגת הארי — War Rights Guide

מדריך אזרחי מקיף לישראלים שנפגעו במסגרת מבצע שאגת הארי (מלחמת ישראל-איראן).

## 📋 מה כולל האתר

- **דף הבית** — מידע כללי + אשף אינטראקטיבי לאיתור זכויות
- **פגיעה גופנית** — זכויות נפגעי איבה, ביטוח לאומי, תמיכה נפשית
- **נזק לרכוש ורכב** — קרן פיצויים, ביטוחים, צעדי תביעה
- **עבודה ועסקים** — זכויות עובדים, פיטורים, חל"ת, עסקים שנפגעו
- **היערכות מוקדמת** — רשימת ציוד, תיעוד רכוש, היערכות מוקדמת
- **גורמים מסייעים** — ספריית קשרים עם מסנן וחיפוש

## 🚀 הפעלה על GitHub Pages

### שלב 1 — Upload לGitHub

```bash
git init
git add .
git commit -m "Initial commit - Roaring Lion War Guide"
git branch -M main
git remote add origin https://github.com/USERNAME/roaring-lion.git
git push -u origin main
```

### שלב 2 — הפעלת GitHub Pages

1. כנס ל-Settings של הרפוזיטורי
2. לחץ על **Pages** בתפריט הצד
3. תחת **Source** — בחר `Deploy from a branch`
4. בחר `main` + `/ (root)`
5. לחץ **Save**

האתר יהיה זמין תוך כ-2 דקות בכתובת:
`https://USERNAME.github.io/roaring-lion/`

## ✨ תכונות עיקריות

- **אשף אינטראקטיבי** — עץ החלטות המוביל לצעדים מותאמים אישית
- **רשימות תיוג** — עם שמירה ב-localStorage
- **חיפוש ומסנן** — בספריית הגורמים המסייעים
- **עיצוב Dark Mode** — עם תמיכה מלאה ב-RTL עברית
- **מותאם לנייד** — תפריט המבורגר, עמודות רספונסיביות
- **ציר זמן** — מה לעשות ומתי אחרי פגיעה
